# डिप्लॉयमेंट और कॉन्फ़िगरेशन

## एनवायरनमेंट वेरिएबल्स

- DISCORD_TOKEN: बॉट प्रमाणीकरण टोकन।
- FIREBASE_CONFIG: डेटाबेस क्रेडेंशियल्स।
- SPE_USER_ID: विशेष डेवलपर उपयोगकर्ता ID।
- NODE_ENV: प्रोडक्शन या डेवलपमेंट मोड।

## स्टार्टअप स्क्रिप्ट

- start.bat: Windows स्टार्टअप स्क्रिप्ट।
- start.sh: Linux स्टार्टअप स्क्रिप्ट।
- पैकेज मैनेजर: डिपेंडेंसी मैनेजमेंट के लिए pnpm का उपयोग करता है।

## मॉनिटरिंग

- स्वास्थ्य जांच: स्थिति के लिए HTTP एंडपॉइंट।
- लॉगिंग: रोटेशन और आर्काइविंग के साथ फाइल-आधारित।
- आंकड़े: Firebase में उपयोग मेट्रिक्स ट्रैक किए जाते हैं।

## बैकअप सिस्टम

- स्थानीय: स्थानीय रूप से संग्रहीत संपीड़ित JSON फाइलें।
- रिमोट: webhook के माध्यम से डिस्कॉर्ड चैनल को भेजा जाता है।
- अनुसूची: हर 5 मिनट में स्वचालित रूप से चलता है।

## स्केलिंग

- मेमोरी: सिंगल-इंस्टेंस डिप्लॉयमेंट के लिए अनुकूलित।
- डेटाबेस: साझा स्टेट के लिए Firebase का उपयोग किया जाता है।
