# QuranBot परियोजना दस्तावेज़ीकरण

## अवलोकन

QuranBot डिस्कॉर्ड प्लेटफॉर्म के लिए डिज़ाइन किया गया एक व्यापक Node.js एप्लिकेशन है। इसका प्राथमिक उद्देश्य कुरान तिलावत, रेडियो स्ट्रीमिंग, अज़कार ऑटोमेशन और नमाज़ के समय की जानकारी सहित इस्लामी सामग्री सेवाएं प्रदान करना है। परियोजना उच्च उपलब्धता, स्थिरता और मॉड्यूलर स्केलेबिलिटी के लिए इंजीनियरिंग दृष्टिकोण के साथ विकसित की गई है।

## लक्षित दर्शक

- इस्लामी सामग्री ऑटोमेशन की तलाश करने वाले डिस्कॉर्ड सर्वर व्यवस्थापक।
- मॉड्यूलर डिस्कॉर्ड बॉट वास्तुकला में रुचि रखने वाले डेवलपर्स।
- विश्वसनीय ऑडियो स्ट्रीमिंग और धार्मिक उपयोगिताओं की आवश्यकता वाले अंतिम उपयोगकर्ता।

## मुख्य विशेषताएं

- ऑडियो प्लेबैक: कई क़ारियों और रेडियो स्टेशनों से कुरान तिलावत स्ट्रीमिंग।
- स्थिरता: Firebase के माध्यम से सर्वर कॉन्फ़िगरेशन को सहेजना और पुनर्स्थापित करना।
- स्वचालित अज़कार: अनुस्मारक संदेशों की अनुसूचित भेजना।
- नमाज़ के समय: स्थान-आधारित सटीकता के साथ वैश्विक नमाज़ समय खोज।
- व्यवस्थापक नियंत्रण: सर्वर प्रबंधन के लिए समर्पित डेवलपर पैनल।
- पुनर्प्राप्ति सिस्टम: रीस्टार्ट के बाद वॉइस चैनल में स्वचालित पुनः कनेक्शन।

## इंस्टॉलेशन

1. रिपॉजिटरी को क्लोन करें।
2. pnpm install का उपयोग करके डिपेंडेंसी इंस्टॉल करें।
3. .env में एनवायरनमेंट वेरिएबल्स कॉन्फ़िगर करें।
4. pnpm start का उपयोग करके बॉट शुरू करें।
