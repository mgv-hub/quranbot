# የተለዩ ዩቲሊቲዎች

## Envira ጫኚ

በ core/package/Envira/ ውስጥ ይገኛል።

- ዓላማ፡ ከኢንክሪፕሽን ድጋፍ ጋር ለ dotenv የተለየ ተለዋጭ።
- ባህሪያት፡ .env ፋይሎችን ይተረጉማል፣ የተጠበቁ ዋጋዎችን ይደግፋል፣ ብዙ አካባቢዎችን ይይዛል።

## የመንገድ ስም

- ላይብረሪ፡ pathlra-aliaser።
- አጠቃቀም፡ እንደ @logger ያሉ ኢምፖርቶችን ከተዛማጅ መንገዶች ይልቅ ይፈቅዳል።
- ቅንብር፡ በ package.json ውስጥ በ path*aliaser* ስር ተገልጿል።

## የድም፝ ዩቲሊቲዎች

- የእንደገና ሙከራ ሎጂክ፡ fetchWithRetry የኔትወርክ አለመረጋጋትን ይይዛል።
- የስትሪም ማረጋገጫ፡ ከማጫወት በፊት የይዘት አይነትን እና ሁኔታን ያረጋግጣል።
- የርዝመት ስሌት፡ ለፕሮግረስ ክትትል የድም፝ ርዝመትን ይገምታል።

## የዳታቤዝ ክሊነር

- ተግባር፡ ቦቱ እንደገና በሌለባቸው ጊልዶች ላይ አሮጌ ዳታን ያጠፋል።
- ግቦች፡ ሴትአፕ ጊልዶች፣ የጊልድ ሁኔታዎች እና የቁጥጥር መለያዎች።
