# የትዕዛዝ ስርዓት

## ስላሽ ትዕዛዞች

- /control: ለማጫወት ዋናውን የቁጥጥር ፓነል ያሳያል።
- /setup: የቁርአን ክፍል እና ቻናሎችን ይፈጥራል።
- /join: ከተዘጋጀው የድምፅ ቻናል ጋር ይገናኛል።
- /leave: ከድምፅ ቻናል ይገለላል።
- /ping: የቦት መዘግየት እና ስታቲስቲክስን ያሳያል።
- /prayer_times: ለተመረጡ ቦታዎች የጸሎት ሰዓቶችን ያሳያል።
- /sources: ቦቱ የሚጠቀምባቸውን የዳታ ምንጮች ይዘርዝራል።
- /guide: የአጠቃቀም መመሪያዎችን ያሳያል።

## የፈቃድ ደረጃዎች

- አስተዳዳሪ፡ ለሁሉም ትዕዛዞች ሙሉ መድረስ።
- ልዩ ተጠቃሚዎች፡ በአካባቢ ተለዋዋጮች ውስጥ የተገለጹ።
- ሁሉም፡ በተለየ ሁኔታዎች ውስጥ ለማጫወት ብቻ የተገደበ።

## የጊዜ ገደቦች

ትዕዛዞች ጥቅም ላይ አለመዋልን ለመከልከል የጊዜ ገደብ ስርዓትን ይጠቀማሉ።

- የተጠቃሚ ጊዜ ገደቦች፡ ለእያንዳንዱ ተጠቃሚ ለእያንዳንዱ ትዕዛዝ ይተገበራል።
- የሰርቨር ጊዜ ገደቦች፡ ለሴትአፕ ትዕዛዞች ለእያንዳንዱ ጊልድ ይተገበራል።
- ዓለም አቀፍ ጊዜ ገደቦች፡ በከፍተኛ ጫና ሁኔታዎች ውስጥ ይተገበራል።

## የአፈጻጸም ፍሰት

1. ኢንተራክሽን ተቀብሏል።
2. የፈቃዝ ማረጋገጫ።
3. የጊዜ ገደብ ማረጋገጫ።
4. የትዕዛዝ አፈጻጸም።
5. የሁኔታ ማዘመን።
