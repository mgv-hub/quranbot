# የመጀመሪያ እና የማስጀመሪያ ፍሰት

## ቅደም ተከተል

1. የአካባቢ ጫኚ፡ core/config/envSwitcher.js የአካባቢ ተለዋዋጮችን ይጭናል።
2. የክላይንት ማስጀመሪያ፡ core/startup/botSetup.js የዲስኮርድ ክላይንት እና ግሎባል ተለዋዋጮችን ያስጀምራል።
3. የዳታ ጫኚ፡ core/data/data-manager.js የቁርአን እና የተቀናይ ዳታን ያመጣል።
4. መግቢያ፡ ቦቱ ወደ ዲስኮርድ ይገባል።
5. ዝግጁ ክስተት፡ core/startup/readyHandler.js ይጀምራል።
   - ፋየርቤዝን ያስጀምሩ።
   - የሥራ ሰዓት ሁኔታዎችን ይመልሱ።
   - የድም፝ ግንኙነቶችን ያግኙ።
   - ትዕዛዞችን ይመዝግቡ።
   - ጊዜ መለኪያዎችን ይጀምሩ (አዝካር፣ ጀርባ ኮፒ፣ ስታቲስቲክስ)።

## የሚደጋገሙ ተግባራት

- የሁኔታ መቀመጥ፡ በየ60 ሰከንዱ።
- ጀርባ ኮፒ፡ በየ5 ደቂቃው።
- የሬዲዮ ጤና፡ በየ30 ደቂቃው።
- የሜሞሪ ማጽዳት፡ በየ3 ደቂቃው።
- የስታቲስቲክስ ማዘመን፡ በየ10 ሰከንዱ።
