# የቁርአንቦት ፕሮጀክት ሰነድ

## አጠቃላይ እይታ

ቁርአንቦት ለዲስኮርድ መድረክ የተዘጋጀ ሰፊ የኖድ.ጂ.ኤስ መተግበሪያ ነው። ዋናው ዓላማው የቁርአን ንባብ፣ የሬዲዮ ስርጭት፣ የአዝካር አውቶሜሽን እና የጸሎት ሰዓት መረጃን ጨምሮ የእስልምና ይዘት አገልግሎቶችን ማቅረብ ነው። ፕሮጀክቱ ከፍተኛ ተደራሽነት፣ የሁኔታ መጠበቅ እና ሞጁላር ስኬል ለማድረግ በምህንድስና አቀራረብ የተሰራ ነው።

## የታለመ ተመልካች

- የእስልምና ይዘት አውቶሜሽን የሚፈልጉ የዲስኮርድ ሰርቨር አስተዳዳሪዎች።
- ሞጁላር የዲስኮርድ ቦት አርክቴክቸር ትኩረት የሚሰጡ ገንቢዎች።
- አስተማማኝ የድምፅ ስርጭት እና የሃይማኖት መሳሪያዎች የሚያስፈልጋቸው የመጨረሻ ተጠቃሚዎች።

## ዋና ባህሪያት

- የድምፅ ማጫወት፡ ከብዙ ተቀናዮች እና የሬዲዮ ጣቢያዎች የቁርአን ንባብ ስርጭት።
- የሁኔታ መጠበቅ፡ በፋየርቤዝ በኩል የሰርቨር ቅንብሮችን ማከማቸት እና መመለስ።
- አውቶሜትድ አዝካር፡ የመታሰቢያ መልእክቶች በተያዘ ሰዓት መላክ።
- የጸሎት ሰዓቶች፡ በቦታ ትክክለኛነት ያለው ዓለም አቀፍ የጸሎት ሰዓት ፍለጋ።
- የአስተዳዳሪ ቁጥጥር፡ ለሰርቨር አስተዳደር የተሰጠ የገንቢ ፓነል።
- የመልሶ ማቋቋሚያ ስርዓቶች፡ ከዳግም ጀመር በኋላ ወደ ድምፅ ቻናሎች በራስ ሰር መገናኘት።

## ጫና

1. ሬፖዚቶሪውን ክሎን ያድርጉ።
2. ጥገኝነቶችን በ pnpm install በመጠቀም ይጫኑ።
3. የአካባቢ ተለዋዋጮችን በ .env ያስተካክሉ።
4. ቦቱን በ pnpm start ይጀምሩ።
