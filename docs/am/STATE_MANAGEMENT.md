# የሁኔታ አስተዳደር

## የጊልድ ሁኔታ

እያንዳንዱ ጊልድ በሜሞሪ ውስጥ የተከማቸ የሥራ ሰዓት ሁኔታ ነገር አለው።

- ግንኙነት፡ የድምፅ ግንኙነት ነገር።
- ማጫወት፡ የአሁኑ ሱራ፣ ተቀናይ እና ሁኔታ።
- ጊዜ መለኪያዎች፡ የአዝካር እና የእንቅስቃሴ አለመኖር ጊዜ መለኪያዎች።
- ቅንብር፡ የቁጥጥር ሁኔታ እና የቻናል መለያዎች።

## ተጠባባቂ ሁኔታ

ሁኔታው ከፋየርቤዝ ሪልታይም ዳታቤዝ ጋር ይመጣጠናል።

- መልሶ ማቋቋም፡ ቦቱ እንደገና ሲጀመር ሁኔታዎች ይመለሳሉ።
- የጀርባ ኮፒ፡ በየ5 ደቂቃው አውቶሜትድ የጀርባ ኮፒ።
- ማጽዳት፡ ለተዉ ጊልዶች አሮጌ ዳታ ይወገዳል።

## የሁኔታ መልሶ ማቋቋም

1. ሁኔታዎችን ከፋየርቤዝ ይጫኑ።
2. የቻናል መኖርን ያረጋግጡ።
3. ወደ ድምፅ ቻናሎች እንደገና ይገናኙ።
4. ከተቻለ ማጫወትን ይቀጥሉ።

## የሜሞሪ አስተዳደር

- ከፍተኛ የሜሞሪ አጠቃቀም ላይ ጋርቤጅ ኮሌክሽን ይጀምራል።
- የኢንተራክሽን ካሼ በየጊዜው ይሰረዛል።
- የተጣሱ ግንኙነቶች በራስ ሰር ይጸዳሉ።
