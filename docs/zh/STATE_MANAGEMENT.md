# 状态管理

## 服务器状态

每个服务器都有一个存储在内存中的运行时状态对象。

- 连接：语音连接对象。
- 播放：当前章节、诵读者和模式。
- 计时器：赞念和不活动计时器。
- 配置：控制模式和频道 ID。

## 持久状态

状态与 Firebase Realtime Database 同步。

- 恢复：机器人重启时恢复状态。
- 备份：每 5 分钟自动备份。
- 清理：为离开的服务器删除旧数据。

## 状态恢复

1. 从 Firebase 加载状态。
2. 验证频道存在。
3. 重新连接语音频道。
4. 如果适用，恢复播放。

## 内存管理

- 高内存使用时触发垃圾回收。
- 交互缓存定期清除。
- 销毁的连接自动清理。
