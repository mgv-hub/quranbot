# 命令系统

## 斜杠命令

- /控制：显示播放的主控制面板。
- /设置：创建古兰经类别和频道。
- /加入：加入配置的语音频道。
- /离开：断开语音频道连接。
- /延迟：显示机器人延迟和统计信息。
- /祈祷时间：显示选定位置的祈祷时间。
- /来源：列出机器人使用的数据来源。
- /指南：显示使用说明。

## 权限级别

- 管理员：完全访问所有命令。
- 特殊用户：在环境变量中定义。
- 所有人：在特定模式下仅限于播放导航。

## 冷却时间

命令使用冷却系统来防止滥用。

- 用户冷却：每个用户每个命令应用。
- 服务器冷却：每个服务器为设置命令应用。
- 全局冷却：在高负载情况下应用。

## 执行流程

1. 收到交互。
2. 权限检查。
3. 冷却检查。
4. 命令执行。
5. 状态更新。
