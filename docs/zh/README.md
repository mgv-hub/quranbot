# QuranBot 项目文档

## 概述

QuranBot 是为 Discord 平台设计的综合性 Node.js 应用程序。其主要目的是提供伊斯兰内容服务，包括古兰经诵读、广播流媒体、自动赞念和祈祷时间信息。该项目经过工程设计，具有高可用性、状态持久性和模块化可扩展性。

## 目标受众

- 寻求伊斯兰内容自动化的 Discord 服务器管理员。
- 对模块化 Discord 机器人架构感兴趣的开发者。
- 需要可靠音频流媒体和宗教工具的终端用户。

## 主要功能

- 音频播放：从多位诵读者和广播电台流媒体古兰经诵读。
- 状态持久性：通过 Firebase 保存和恢复服务器配置。
- 自动赞念：定时发送纪念消息。
- 祈祷时间：基于位置的全球祈祷时间查询。
- 管理员控制：专为服务器管理设计的开发者面板。
- 恢复系统：重启后自动重新连接到语音频道。

## 安装

1. 克隆仓库。
2. 使用 pnpm install 安装依赖项。
3. 在 .env 中配置环境变量。
4. 使用 pnpm start 启动机器人。
