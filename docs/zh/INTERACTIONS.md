# 交互处理器

## 处理器

位于 core/interactions/ 中。

- interactionProcessor.js: 所有交互的主路由器。
- proc-buttons.js: 将按钮点击路由到特定逻辑文件。
- proc-menus.js: 处理选择菜单交互。
- proc-modals.js: 处理模态提交。
- proc-commands.js: 执行斜杠命令。

## 系统逻辑

- 语音状态：在允许播放命令之前检查机器人是否在语音频道中。
- 授权：在执行敏感操作之前验证用户权限。
- 冷却：执行速率限制以防止滥用。
- 错误处理：集中式错误捕获，带有用户友好的消息。

## 特定处理器

- 播放：播放、暂停、恢复、下一个、上一个。
- 导航：章节列表和诵读者列表的分页。
- 管理员：仅限开发者的服务器管理和统计控制。
- Webhook: 外部 webhook 赞念服务的注册和管理。
