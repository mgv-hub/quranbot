# 数据管理

## 获取

位于 core/data/ 中。

- 来源：数据从 mp3quran.net、aladhan.com 和自定义 JSON 存储库获取。
- 方法：使用带有自定义头部和超时的 node-fetch。
- 验证：数据结构在加载到全局状态之前进行验证。

## 缓存

- 运行时缓存：数据存储在全局变量中以便快速访问。
- Firebase 缓存：关键数据备份到 Firebase 以进行持久性。
- 本地缓存：如果远程源失败，则使用 JSON 文件作为备用。

## 存储

- Firebase: 用于服务器设置、状态、控制 ID 和用户 webhook。
- 本地：备份文件在发送到 Discord 频道之前被压缩并本地存储。
- 环境：敏感配置通过自定义 Envira 加载器加载。
