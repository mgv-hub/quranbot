# Документация проекта QuranBot

## Обзор

QuranBot - это комплексное приложение Node.js, разработанное для платформы Discord. Его основная цель - предоставление услуг исламского контента, включая чтение Корана, радиотрансляции, автоматизацию зикров и информацию о времени молитв. Проект разработан с учетом высокой доступности, постоянства состояния и модульной масштабируемости.

## Целевая аудитория

- Администраторы серверов Discord, ищущие автоматизацию исламского контента.
- Разработчики, заинтересованные в модульной архитектуре ботов Discord.
- Конечные пользователи, нуждающиеся в надежной аудиотрансляции и религиозных утилитах.

## Основные функции

- Воспроизведение аудио: Трансляция чтений Корана от нескольких чтецов и радиостанций.
- Постоянство состояния: Сохранение и восстановление конфигураций сервера через Firebase.
- Автоматические зикры: Запланированная отправка сообщений поминания.
- Время молитв: Глобальный поиск времени молитв с точностью по местоположению.
- Контроль администратора: Специальная панель разработчика для управления сервером.
- Системы восстановления: Автоматическое переподключение к голосовым каналам после перезапусков.

## Установка

1. Клонируйте репозиторий.
2. Установите зависимости с помощью pnpm install.
3. Настройте переменные окружения в .env.
4. Запустите бота с помощью pnpm start.
