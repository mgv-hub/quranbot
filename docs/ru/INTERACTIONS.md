# Обработчики взаимодействий

## Процессоры

Расположены в core/interactions/.

- interactionProcessor.js: Главный маршрутизатор для всех взаимодействий.
- proc-buttons.js: Направляет нажатия кнопок к конкретным файлам логики.
- proc-menus.js: Обрабатывает взаимодействия меню выбора.
- proc-modals.js: Обрабатывает отправки модальных окон.
- proc-commands.js: Выполняет слэш-команды.

## Логика системы

- Голосовое состояние: Проверяет, находится ли бот в голосовом канале перед разрешением команд воспроизведения.
- Авторизация: Проверяет разрешения пользователя перед выполнением чувствительных действий.
- Задержки: Применяет ограничение скорости для предотвращения злоупотреблений.
- Обработка ошибок: Централизованный перехват ошибок с удобными для пользователя сообщениями.

## Специфические обработчики

- Воспроизведение: Play, pause, resume, next, previous.
- Навигация: Постраничность для списков сур и списков чтецов.
- Администратор: Только для разработчика управление сервером и статистикой.
- Webhooks: Регистрация и управление внешними webhook сервисами зикров.
