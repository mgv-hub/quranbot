# Поток запуска и инициализации

## Последовательность

1. Загрузка окружения: core/config/envSwitcher.js загружает переменные окружения.
2. Инициализация клиента: core/startup/botSetup.js инициализирует клиент Discord и глобальные переменные.
3. Загрузка данных: core/data/data-manager.js получает данные Корана и чтецов.
4. Вход: Бот входит в Discord.
5. Событие готовности: core/startup/readyHandler.js запускается.
   - Инициализировать Firebase.
   - Восстановить состояния времени выполнения.
   - Восстановить голосовые подключения.
   - Зарегистрировать команды.
   - Запустить таймеры (зикры, резервные копии, статистика).

## Повторяющиеся задачи

- Сохранение состояния: Каждые 60 секунд.
- Резервное копирование: Каждые 5 минут.
- Здоровье радио: Каждые 30 минут.
- Очистка памяти: Каждые 3 минуты.
- Обновление статистики: Каждые 10 секунд.
