# Развертывание и конфигурация

## Переменные окружения

- DISCORD_TOKEN: Токен аутентификации бота.
- FIREBASE_CONFIG: Учетные данные базы данных.
- SPE_USER_ID: Специальные ID пользователей разработчиков.
- NODE_ENV: Режим производства или разработки.

## Скрипты запуска

- start.bat: Скрипт запуска Windows.
- start.sh: Скрипт запуска Linux.
- Менеджер пакетов: Использует pnpm для управления зависимостями.

## Мониторинг

- Проверка здоровья: HTTP конечная точка для статуса.
- Логирование: На основе файлов с ротацией и архивированием.
- Статистика: Метрики использования отслеживаются в Firebase.

## Система резервного копирования

- Локальное: Сжатые JSON файлы хранятся локально.
- Удаленное: Отправляется на канал Discord через webhook.
- Расписание: Запускается каждые 5 минут автоматически.

## Масштабирование

- Память: Оптимизировано для развертывания одного экземпляра.
- База данных: Firebase используется для общего состояния.
