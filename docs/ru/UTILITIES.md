# Пользовательские утилиты

## Загрузчик Envira

Расположен в core/package/Envira/.

- Назначение: Пользовательская альтернатива dotenv с поддержкой шифрования.
- Функции: Парсит файлы .env, поддерживает зашифрованные значения, обрабатывает несколько окружений.

## Псевдонимы путей

- Библиотека: pathlra-aliaser.
- Использование: Позволяет импорты типа @logger вместо относительных путей.
- Конфигурация: Определено в package.json под path*aliaser*.

## Аудио утилиты

- Логика повторных попыток: fetchWithRetry обрабатывает нестабильность сети.
- Проверка потока: Проверяет тип контента и статус перед воспроизведением.
- Расчет длительности: Оценивает длительность аудио для отслеживания прогресса.

## Очиститель базы данных

Расположен в core/utils/databaseCleaner.js.

- Функция: Удаляет устаревшие данные для серверов, где бот больше не находится.
- Цели: Серверы настройки, состояния серверов и ID управления.
