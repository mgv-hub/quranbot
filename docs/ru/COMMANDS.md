# Система команд

## Слэш-команды

- /control: Отображает основную панель управления для воспроизведения.
- /setup: Создает категорию Корана и каналы.
- /join: Присоединяется к настроенному голосовому каналу.
- /leave: Отключается от голосового канала.
- /ping: Отображает задержку бота и статистику.
- /prayer_times: Показывает время молитв для выбранных местоположений.
- /sources: Перечисляет источники данных, используемые ботом.
- /guide: Отображает инструкции по использованию.

## Уровни разрешений

- Администратор: Полный доступ ко всем командам.
- Специальные пользователи: Определены в переменных окружения.
- Все: Ограничено навигацией воспроизведения в определенных режимах.

## Задержки

Команды используют систему задержек для предотвращения злоупотреблений.

- Задержки пользователя: Применяются для каждого пользователя для каждой команды.
- Задержки сервера: Применяются для каждого сервера для команд настройки.
- Глобальные задержки: Применяются во время ситуаций высокой нагрузки.

## Поток выполнения

1. Получено взаимодействие.
2. Проверка разрешений.
3. Проверка задержки.
4. Выполнение команды.
5. Обновление состояния.
