# النشر والتكوين

## متغيرات البيئة

- DISCORD_TOKEN: رمز مصادقة البوت.
- FIREBASE_CONFIG: بيانات اعتماد قاعدة البيانات.
- SPE_USER_ID: معرفات مستخدمي المطورين الخاصين.
- NODE_ENV: وضع الإنتاج أو التطوير.

## نصوص بدء التشغيل

- start.bat: نص بدء تشغيل Windows.
- start.sh: نص بدء تشغيل Linux.
- مدير الحزم: يستخدم pnpm لإدارة التبعيات.

## المراقبة

- فحص الصحة: نقطة نهاية HTTP للحالة.
- التسجيل: قائم على الملفات مع التدوير والأرشفة.
- الإحصائيات: مقاييس الاستخدام تُتتبع في Firebase.

## نظام النسخ الاحتياطي

- محلي: ملفات JSON مضغوطة مخزنة محلياً.
- بعيد: تُرسل إلى قناة Discord عبر webhook.
- الجدول: يعمل كل 5 دقائق تلقائياً.

## التوسع

- الذاكرة: محسّنة للنشر أحادي المثيل.
- قاعدة البيانات: Firebase تُستخدم للحالة المشتركة.
