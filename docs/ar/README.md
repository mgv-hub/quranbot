# وثائق مشروع QuranBot

## نظرة عامة

QuranBot هو تطبيق Node.js شامل مصمم لمنصة Discord. الغرض الأساسي منه هو تقديم خدمات المحتوى الإسلامي بما في ذلك تلاوة القرآن، وبث الإذاعات، وأتمتة الأذكار، ومعلومات مواقيت الصلاة. تم هندسة المشروع لتوفر عالي، واستمرارية الحالة، وقابلية التوسع النمطية.

## الجمهور المستهدف

- مسؤولو سيرفرات Discord الباحثون عن أتمتة المحتوى الإسلامي.
- المطورون المهتمون بهندسة بوتات Discord النمطية.
- المستخدمون النهائيون الذين يحتاجون إلى بث صوتي موثوق وأدوات دينية.

## الميزات الرئيسية

- تشغيل الصوت: بث تلاوات القرآن من قراء متعددين ومحطات إذاعية.
- استمرارية الحالة: حفظ واستعادة إعدادات السيرفر عبر Firebase.
- الأذكار التلقائية: إرسال رسائل الذكر المجدولة تلقائياً.
- مواقيت الصلاة: بحث عالمي عن مواقيت الصلاة مع دقة تعتمد على الموقع.
- تحكم المسؤولين: لوحة مطور مخصصة لإدارة السيرفر.
- أنظمة الاستعادة: إعادة الاتصال التلقائي بالقنوات الصوتية بعد إعادة التشغيل.

## التثبيت

1. استنسخ المستودع.
2. ثبت التبعيات باستخدام pnpm install.
3. اضبط متغيرات البيئة في .env.
4. شغل البوت باستخدام pnpm start.
