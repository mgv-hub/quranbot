# إدارة البيانات

## الجلب

موجود في core/data/.

- المصادر: البيانات تُجلب من mp3quran.net، aladhan.com، ومستودعات JSON مخصصة.
- الطرق: يستخدم node-fetch مع ترويسات مخصصة و مهلات زمنية.
- التحقق: هياكل البيانات تُتحقق منها قبل تحميلها في الحالة العامة.

## التخزين المؤقت

- ذاكرة التخزين المؤقت لوقت التشغيل: البيانات تُخزن في متغيرات عامة للوصول السريع.
- ذاكرة التخزين المؤقت Firebase: البيانات الحرجة تُنسخ احتياطياً إلى Firebase للاستمرارية.
- ذاكرة التخزين المؤقت المحلية: تُستخدم ملفات JSON كبديل إذا فشلت المصادر البعيدة.

## التخزين

- Firebase: يُستخدم لإعدادات السيرفر، والحالات، ومعرفات التحكم، وويب هوك المستخدمين.
- محلي: ملفات النسخ الاحتياطي تُضغط وتُخزن محلياً قبل إرسالها إلى قنوات Discord.
- البيئة: التكوين الحساس يُحمل عبر محمل Envira المخصص.
